# Tip Calculator

A bill splitter and tip calculator web app built for the Dev Weekends Fellowship 2026 assessment.

## How to Run

1. Download or clone this repository
2. Open `index.html` in any browser
3. That's it — no installation needed!

## Features

- Live tip calculation as you type
- Preset tip buttons (10% / 15% / 20% / Custom)
- Bill splitting across multiple people
- Inline validation for all inputs
- Reset button
- Responsive design for mobile and desktop
